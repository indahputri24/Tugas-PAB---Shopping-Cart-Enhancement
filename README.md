Halaman Checkout digunakan untuk menyelesaikan proses pemesanan produk yang ada di cart.

#### - Form

<img width="3200" height="2000" alt="Cuplikan layar 2026-02-24 095449" src="https://github.com/user-attachments/assets/acd023fc-7496-4df8-861a-16f16461cc2c" />

- Nomor HP
